GIPHER APPLICATION

Gipher is an application where the user can see a list of trendig gifs. These gifs are fetched from an external api. The user can select a gif and add it to his 'Favourites' list. He can view his favourites list and can give some comment to the gifs present there. He can also delet any gif from that list. In addition, the user can search gifs according to his need and add them to his favourites. The front-end was developed using material-UI on angular. The backend was developed using Spring-Boot microservices and restful API. Database language used was MYSQL. Frontend unit testing done using Karma. Backend unit testing done using Mock-mvc and Mockito. End-to-end testig done using Protractor.
